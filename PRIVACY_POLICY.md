# Privacy Policy

**Adult Domain Detector (AI)** respects user privacy.

## Data Collection
- This extension does **not collect, store, or transmit** personal data.
- No browsing activity is tracked.

## User Input
- Text pasted by the user is analyzed only when the user clicks "Analyze".
- If AI is enabled, the pasted text is sent directly to the selected AI provider.

## API Keys
- API keys are provided by the user.
- Keys are stored securely using Chrome storage.
- Keys are never shared or logged.

## Third-Party Services
Depending on user choice, the extension may interact with:
- OpenAI
- Groq
- Google Gemini

These services process data according to their own privacy policies.

## Changes
This policy may be updated if features change.

## Contact
For questions or concerns, please open an issue on GitHub.
